# WebSE
<img src="https://github.com/Happy-Lotus/WebSE/blob/master/photo/webse.png" width="100%">
2023-1 컴퓨터공학기초캡스톤디자인 - HTML, CSS, JavaScript를 활용한 프로젝트입니다.
<br>

## 프로젝트 소개 & 진행 기간

- **프로젝트 소개**<br>
공격 시나리오를 사이버 킬체인에 기반하여 작성하고 이를 MITER ATT&CK에 매핑해 웹사이트의 취약점을 분석하는 최적의 솔루션 제공 서비스를 기획했습니다. 웹 해킹 트렌드를 설명하고 공격 시나리오를 통한 연구를 통해 다른 서비스와는 차별화된 전문화된 서비스를 제공합니다. 개발한 프레임워크를 통해 고객의 정보자산을 안전하게 보호합니다. 해당 사이트는 자신의 시스템에 어떤 취약점이 있는지 알고 싶어하는 의뢰인들의 문의를 받고 우리 서비스를 사람들에게 알리기 위해 개발하였습니다. 

- **진행 기간**<br>
  2022.04~2022.6
<br>

## 개발배경
랜섬웨어가 증가하는 가운데  중소기업과 소상공인의 피해가 대부분을 차지하는 것을 확인할 수 있었습니다. 재정적인 어려움과 한정된 자원, 노하우의 부족 등 이유로 정보 보안에 투자하지 않고 있었습니다. 그래서 보고서를 통해 공격 시나리오를 작성하고 시각적으로 보여줌으로써 소상공인들이 쉽게 랜섬웨어 감염을 사전에 방지할 수 있는 서비스가 필요하다고 생각하여 개발하게 되었습니다.
